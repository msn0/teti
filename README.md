## Installation

```
npm i -g teti
```

## Usage

```
$ teti --help

Usage
  $ teti <url>

Options
  -n          number of tests to run (10 is default)
  --runner    chrome or phantom (chrome is default, at least chrome 59 is needed)
  --verbose   output all data
```
